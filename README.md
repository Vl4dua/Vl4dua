## In The Spotlight

### StreamShield

**StreamShield** is a privacy-focused browser extension made for streamers and creators who want to browse live without exposing sensitive information on screen.

It protects things like:
- IP addresses
- emails
- phone numbers
- addresses
- geo and ISP data
- user-agent strings
